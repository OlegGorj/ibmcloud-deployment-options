Textbeschreibung für container_group_autorecovery.gif

Es werden drei blaue Felder angezeigt, die drei Instanzen in einer Containergruppe darstellen.

Das erste Feld wird schwarz. Dies bedeutet, dass die erste Instanz nicht mehr verfügbar ist.

Das erste schwarze Feld wird ausgeblendet.

An der Position, an der sich das erste Feld befand, wird ein anderes blaues Feld angezeigt. Das bedeutet, dass die alte Containerinstanz durch eine neue Instanz ersetzt wurde.
