Textbeschreibung für container_group_loadbalancing.gif

Eine Person wird neben einem grauen Feld mit der Bezeichnung A angezeigt. Die Person stellt einen Benutzer und Feld A eine Containergruppe dar.

Innerhalb von Feld A wird ein blaues Feld mit der Bezeichnung B angezeigt, das die Lastausgleichsfunktion für die Containergruppe darstellt.

Außerdem werden drei weitere Felder in Feld A angezeigt, die drei Instanzen in der Containergruppe darstellen.

Eine Anforderung wird vom Benutzer über die Lastausgleichsfunktion an die erste Containergruppeninstanz gesendet. Diese Anforderung dreht sich und wird - wenn sie abgeschlossen ist - ausgeblendet.

Eine Kommunikation wird vom Benutzer über die Lastausgleichsfunktion an die zweite Instanz gesendet. Diese Anforderung dreht sich und wird nicht abgeschlossen. Die Anforderung dreht sich weiter.

Eine Kommunikation wird vom Benutzer über die Lastausgleichsfunktion an die dritte Instanz gesendet. Diese Anforderung dreht sich und wird - wenn sie abgeschlossen ist - ausgeblendet.

Eine Anforderung wird vom Benutzer über die Lastausgleichsfunktion an die erste Instanz gesendet. Diese Anforderung dreht sich und wird - wenn sie abgeschlossen ist - ausgeblendet.

Eine Kommunikation wird vom Benutzer über die Lastausgleichsfunktion an die zweite Instanz gesendet. Diese Anforderung dreht sich und wartet, bis die hängende Anforderung abgeschlossen ist. Wenn die hängende Anforderung abgeschlossen und ausgeblendet wird, dreht sich die nächste Anforderung und wird - wenn sie abgeschlossen ist - ausgeblendet.

Eine Kommunikation wird vom Benutzer über die Lastausgleichsfunktion an die dritte Instanz gesendet. Diese Anforderung dreht sich und wird - wenn sie abgeschlossen ist - ausgeblendet.
