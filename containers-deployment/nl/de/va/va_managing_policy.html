Organisationsrichtlinien überprüfen

Als Manager einer Bluemix-Organisation können Sie eine Richtlinie festlegen, die entweder Benutzer warnt oder die Bereitstellung von Containern aus Images mit bestimmten Sicherheits- und Konfigurationsproblemen blockiert.

Der Organisationsmanager kann bestimmen, ob Images in Ihrer Organisation bereitgestellt werden können, wenn ein Sicherheits- oder Konfigurationsproblem gemeldet wird:
  • Wenn ein bestimmtes Sicherheits- oder Konfigurationsproblem gefunden wird, kann die Bereitstellung eines Containers aus dem Image blockiert werden.
  • Benutzer können vor einem bestimmten Sicherheits- oder Konfigurationsproblem gewarnt werden, aber gleichzeitig immer noch die Erlaubnis erhalten, einen Container aus dem Image bereitzustellen.

Standardmäßig lassen die organisationsbezogenen Richtlinien die Bereitstellung von Images mit Warnungen zu. Richtlinien für Container können nicht geändert werden und werden von Bluemix zur Gewährleistung der Sicherheit aller Containerbenutzer in der Cloud bereitgestellt.

Mit den folgenden Schritten legen Sie fest, ob die Bereitstellung von Images mit bestimmten Sicherheits- oder Konfigurationsproblemen zulässig sein oder blockiert werden soll.

  1. Melden Sie sich bei Bluemix an. Die Vulnerability Advisor (Vulnerability Assessment) ist nur in der Benutzerschnittstelle verfügbar, wenn Sie angemeldet sind.

  3. Klicken Sie im Bereich Schwachstellenanalyse auf Richtlinien verwalten. Standardmäßig werden die Richtlinieneinstellung für Images angezeigt.
  4. Überprüfen Sie die Einstellungen für die Bereitstellungsrichtlinien für Images. Die Einstellungen für die Bereitstellungsrichtlinien für Images können nur von Organisationsmanagern geändert werden.
  5. Überprüfen Sie den Vulnerability Advisor-Bericht für alle Container-Images in Ihrer Organisation. Die aufgelisteten Images werden automatisch aktualisiert und so kategorisiert, dass die Änderungen an den Einstellungen für die Bereitstellung wiedergegeben werden.
  6. Überprüfen Sie die Einstellungen für Containerrichtlinien. Entsprechen die in Ihrer Organisation ausgeführten Container nicht den Bluemix-Richtlinien, werden sie in dieser Ansicht aufgelistet.
