Examen des règles de l'organisation

Si vous êtes le responsable de l'organisation Bluemix, vous pouvez définir une règle afin d'avertir les utilisateurs ou de bloquer le déploiement de conteneurs à partir d'images affectées par des problèmes de sécurité ou de configuration spécifiques.

Le responsable de l'organisation peut décréter si des images de votre organisation peuvent être déployées lorsqu'un problème de sécurité ou de configuration est signalé :
  • Si un problème de sécurité ou de configuration est détecté, le déploiement d'un conteneur depuis l'image peut être bloqué.
  • Les utilisateurs peuvent être avertis de la présence d'un problème de sécurité ou de configuration spécifique, tout en étant néanmoins autorisés à déployer un conteneur depuis cette image.

Par défaut, les règles d'organisation autorisent le déploiement d'images concernées par des avertissements. Les règles régissant les conteneurs ne peuvent pas être modifiées et sont fournies par Bluemix pour veiller à la sécurité de tous les utilisateurs de conteneurs dans le cloud.

Pour permettre ou bloquer le déploiement d'images concernées par des problèmes de sécurité ou de configuration spécifique, procédez comme suit.

  1. Connectez-vous à Bluemix. Vulnerability Advisor est disponible dans l'interface utilisateur uniquement lorsque vous êtes connecté.

  3. Dans la section Evaluation des vulnérabilités, cliquez sur Gérer les règles. Par défaut, les paramètres de règles pour Images sont affichés.
  4. Passez en revue les paramètres Règles de déploiement d'image. Seuls les responsables de l'organisation peuvent modifier les paramètres Règles de déploiement d'image.
  5. Passez en revue le rapport Vulnerability Advisor sur toutes les images de conteneur de votre organisation. Les images répertoriées sont automatiquement actualisées, puis classées afin de refléter les modifications apportées aux paramètres de déploiement.
  6. Passez en revue les paramètres Règles de conteneur. Si des conteneurs en exécution dans votre organisation ne se conforment pas à la règle Bluemix, ils sont répertoriés dans cette vue.
